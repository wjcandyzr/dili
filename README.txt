地理地貌 3D 教学展示
====================

【如何打开】
  推荐：双击 start.bat（自动启动本地服务器并打开浏览器）
  备选：直接双击 index.html（部分浏览器可能因 file:// 限制无法加载模块）

【首次使用】
  需要联网加载 Three.js（CDN）。
  第二次之后浏览器会缓存，断网也能打开。

【操作说明】
  - 左侧列表：切换 13 种地貌
  - 鼠标左键拖拽：旋转视角
  - 鼠标滚轮：缩放
  - 鼠标右键拖拽：平移
  - 顶部"3D 视图 / 等高线 / 剖面图"按钮：切换视图模式
  - 顶部"显示标注"：开关地形上的名称浮标
  - 顶部"自动旋转"：一键自动旋转演示（方便上课讲解）
  - 点击地形上的蓝色圆点：查看该地貌要素的详细说明

【收录的 13 种地貌】
  基础陆地：山地、高原、丘陵、平原、盆地
  内外力地貌：火山、峡谷、河流三角洲
  中国特色：喀斯特（岩溶）、丹霞
  外力作用：沙漠、冰川、海岸

【推荐浏览器】
  Chrome、Edge（最新版）
